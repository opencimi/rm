Archetypes act as the configuration basis for the particular structures of instances defined by the reference model. To enable archetypes to be used to create valid data, key classes in the reference model act as  root  points for archetyping. These classes have the archetype_details attribute set. An instance of the class ARCHETYPED contains the relevant archetype identification <br /></p><p>information, allowing archetypes to be matched up with data instances.<br /></p>
